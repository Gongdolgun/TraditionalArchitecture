# Traditional Architecture

### 프로젝트 기간
2021.11 ~ 2022.01

### 프로젝트 내용
Unity로 제작된 한국 전통건축 관련 국가 사업입니다.
영상과 간단한 게임 및 AR 컨텐츠로 여러가지 전통건축 관련 지식을 습득할 수 있는 프로젝트입니다.
